# kenyan-job-site-scrapper
A Python project to automate the collection of job postings from Kenyan job portals, enabling job seekers and recruiters to track opportunities efficiently.


# Overview

This project scrapes job listings from popular Kenyan job websites (e.g., BrighterMonday.co.ke, JobWeb.co.ke, RapidJobs.co.ke) using Python. It extracts key details such as job title, company name, location, job type, posted date, and job description, then saves the data into a structured CSV file for easy analysis.
